# 🎮 Tic Tac Toe Game (Console-Based)

## 📖 Overview

Tic Tac Toe is a console-based game developed in C++ that demonstrates fundamental programming concepts such as arrays, loops, functions, and conditional statements. The game allows two players to play alternately on a dynamic 3×3 board, automatically detects wins and draws, and provides an option to replay the game.

---

## ✨ Features

- 🎮 Two-player gameplay (Player X vs Player O)
- 📋 Dynamic 3×3 game board
- 🔄 Board updates after every move
- ✅ Win detection (Rows, Columns, Diagonals)
- 🤝 Draw detection
- ✔️ Input validation for invalid or occupied positions
- 🔁 Replay option
- 🖥️ Console-based user interface

## 🎯 Game Rules

1. The game is played between two players: **X** and **O**.
2. Players take turns choosing a position from **1 to 9**.
3. The first player to align **three symbols** horizontally, vertically, or diagonally wins.
4. If all positions are filled without a winner, the game ends in a draw.
5. Players can choose to play again after the game ends.
